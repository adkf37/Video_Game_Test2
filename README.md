# Dogsled Kart

Lightweight 3D dogsled racing game built with **Godot 4.x + GDScript**.

## Quick Start
1. Install [Godot 4.3+](https://godotengine.org/download).
2. Open the `godot/` folder as a Godot project.
3. Press **F5** (or Play) — the Main Menu will appear.
4. Click **Play** to drive the sled on a flat plane (WASD / Arrow keys).

## Project Structure
```
godot/           Godot project root
  scenes/        .tscn scene files
  scripts/       .gd script files
  assets/        placeholder art, audio
docs/            Design docs, task packets
```
